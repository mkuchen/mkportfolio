Project Outline

Implement the design specifications for a simplistic example of referral marketing web software. The specifications included:

  • Form submission for new Links to be added by visitors
  • Each Link form has a single, non-null "title" field
  • A Links page where visitors can see all of the Links that have been submitted
  • The Links page has four columns: one for the title, the number of times it's been clicked, an Edit link, and a Delete link.
  • Each Link has its own landing page, located at {your_url}/landing/{link_title}
  • The landing page redirects to {your_url}/landing/?link={link_title}, which has the link title displayed on the page

Some extra features I added were:

  • A user profile service for users to generate accounts
  • A profile image input field for user profiles
  • Periodic tasks for recalculating link click statistics, and historical statistics tracking
  • The use of HighCharts.js to display this historical click data

Sengrid Gateway was designed to be a middle man between a Marketfy and Sendgrid; it would handle the sending of emails from the site to Sendgrid, and the event tracking from Sendgrid to the site. Periodically, the site would request updated statistics from Sendgrid Gateway's API. The statistics that had changed since the last "pull" were sent from Sendgrid Gateway to Marketfy for the drip campaign admins to monitor their progress.

Project Outline

Design an email logging server that keeps track of drip campaign statistics by logging events from the Sendgrid API such as delivery and bounce notices, open notifications, and link-click events. Sendgrid Gateway took advantage of Tornado's long polling capabilities to log the events, in-order, as they were sent from Sendgrid and received by the server.

Project Outline

Redesign the Maize & Blue Fund's website. The fund had been operating from an outdated web platform for several years and wanted a new look for visitors who were looking for information. The project entailed designing backgrounds and logos using GIMP image editor, proposing design choices to the fund's board members, and meeting deadlines jointly set by the board and myself.
